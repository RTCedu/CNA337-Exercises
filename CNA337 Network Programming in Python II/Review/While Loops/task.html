While Loops

There is another Python statement that can also be used to build an iteration. It is called the while statement. The while statement provides a much more general mechanism for iterating. Similar to the if statement, it uses a boolean expression to control the flow of execution. The body of while will be repeated as long as the controlling boolean expression evaluates to True.

Problem

Write a function, get_log_event_list that takes 2 strings, a file_name and an event, and uses a while loop to output a list containing the lines where the event was found:

1. Create an empty list to hold the matching lines

2. Open a the file for reading

3. Use a while loop to check if the file has a next line, or if the next line is blank

4. In the while loop, if the event string is found in the line, add it to the list.

5. In the while loop, read the next line.

6. Outside the while loop, return the list.

Use readline()
