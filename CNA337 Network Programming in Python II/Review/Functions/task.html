Functions

In Python, a function is a named sequence of statements that belong together. Their primary purpose is to help us organize programs into chunks that match how we think about the solution to the problem.

The syntax for a function definition is:

def name( parameters ):
    statements

It is common to use placeholder functions in testing, that behave like the actual function without hardware. These often prefixed with fake, mock, or dummy, such as fake_ping

Problem

1. Write a fake_ping function, that takes a server_ip and a ping_time, and returns a string

2. Have the ping function call fake_ping with a ping time of 10ms, and return the result of the fake_ping function. Have the fake_ping function return a string that says "Ping response from [server_ip] [ping_time]"

As an example, if you wrote ping("127.0.0.1"), the code should return:

Ping response from 127.0.0.1 10ms

server_ip and ping_time are strings.

Both functions, ping and fake_ping, should return a string (str). This means you don't need to use print in your code, though it is fine for testing and debugging.
